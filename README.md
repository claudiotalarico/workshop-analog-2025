#### Klayout - common short-cuts
`Edit > Selection > Flip Horizontally` - __`Shift + H`__<br>
`Edit > Selection > Move by` - __`[`__<br>
`Edit > Selection > Change Layer` - __`Shift + L`__<br>
`Edit > Copy` - __`Ctrl + C`__<br>
`Edit > Cut` - __`Ctrl + X`__<br>
`Edit > Paste` - __`Ctrl + V`__<br>
`Edit > Clear All Rulers and Annotations` - __`Shift + K`__<br>
`Display > Zoom Fit` - __`F`__<br>
`Display > Zoom In` - __`Z`__<br>
`Display > Zoom Out` - __`Shift + Z`__<br>
`Display > Full Hierarchy` - __`Shift + F`__
